# Loops

## **for** loop
- used to compute ranges
- used when the number of iterations is known

```
for(/*initialization*/; /*condition*/; /*operation*/) {
//code block
}
```

```
for(let i = 1; i <=10; i = i+1) {
    console.log(1)
}
```

**Executes**
1. initialization
2. condiiton
3. code block
4. operation
5. repeat from step 2
